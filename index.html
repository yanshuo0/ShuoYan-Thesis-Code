About knowledge fighter

This application is  a master project proposed by Prof. Martin Gander &  Prof. Pierre-Alain Cherix from the Mathematics Department of University of Geneva. The aim of this project is to free players from using predefined data embedded by game designer. We introduce a knowledge importer which allows players to enter their own data and then the game interface will use these user-defined data in the game.

This application is programmed by Shuo Yan, master student in Computer Science at University of Geneva, supervised by Prof. Martin Gander and Prof. Pierre-Alain Cherix. This application is Open-Srouced, all resources could be found on Github. It can be further used as part of other projects.

Instruction of Data Importer

Please input of copy-paste your data in the abover textare and press "Save" button. Click "Reset" button to clear all input.

  1. Each data should be input in a new line.
  2. Each data should be input in the format of "Question = Answer".
  3. After clicking "Save" button, please choose a game to play in the Galerie section by clicking corresponding button.
  4. To play Space Invaders, you need to input 8 data.
